Instrucciones para el manejo del ejemplo dinámico:

Este ejemplo es una breve descripción sobre las funciones universales. El objetivo se este ejemplo es observar el comportamiento de una función universal. Las funciones a simular son las que se encuentras en menú de opciones.
1.Es necesario que se ingrese valores en los dos campos
2.En las opciones de raíz cuadrada, raíz cúbica,valor absoluto, valor trunco, es primo, serie fibonnaci y factorial, se realizan las operaciones correspondientes con el primer valor ingresado.
3.Para la opción de valor trunco se deben ingresar números decimales

Nuestra función universal está definida como:

U(operación,x,y)

En donde:
U es la función universal
operación es la función a simular
x,y son las variables de entrada
